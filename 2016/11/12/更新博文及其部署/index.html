更新博文及其部署

发表于 2016-11-12

新建博文

  • 第一种方法： hexo new “博文题目”
  • 第二种方法： 直接在hexo目录下的source_posts下 新建 博文名.md文件

写博文

直接用markdown编辑器编辑博文。
markdown入门指南

发博文

啦啦啦！终于到最后一步了。只有发表了才能被看到呀！将博客部署到GitHub。

  • hexo clean （清除缓冲）
  • hexo generate （生成静态文件）
  • hexo server （本地浏览）
  • hexo deploy （部署到GitHub）

快捷命令
hexo generate == hexo g
hexo sever == hexo s
hexo deploy == hexo d

刷新你的个人博客，就可以看到新鲜出炉的博文啦！
